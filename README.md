# FusionRL — Reinforcement Learning для управления плазмой в токамаке

**FusionRL** — исследовательский проект по применению обучения с подкреплением для управления плазмой в токамаке.

## О проекте
Управление плазмой — сложная нелинейная задача. Классические регуляторы работают только в узком диапазоне. Мы сравнили алгоритмы PPO и DDPG в симуляторе плазмы.

## Результаты
| Алгоритм | Стабильность | Награда |
|----------|--------------|---------|
| PPO      | 92.3%        | 85.2    |
| DDPG     | 84.6%        | 76.8    |
| PID      | 42.9%        | 42.5    |

## Структура
- `model/` — физическая модель и RL-агенты
- `web/` — веб-симулятор
- `train.py` — обучение
